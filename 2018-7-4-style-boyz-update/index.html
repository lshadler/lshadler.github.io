Installing the game

Windows

Click Here to get started with our awesome game!

Mac

Click Here and unzip the .app file. From there, “right-click” on the .app file and click “Show Package Contents”. Navigate to the Contents/MacOS/ folder and double-click the Style_Boyz_The_Game file. When the dialog opens up, click “Open” (Hopefully you will trust us :D). If no dialog appears, or the dialog displays a notification related to “Damaged files”, you will need to loosen the security settings of your system.
